About

Learning Deserves An Upgrade

Math shouldn't be hard. It's natural, and present in everyday life. However, the classroom can make it hard. That's why I created Todarith. Todarith was created to help students by innovating in the latest technologies in order to better help those with a passion to learn.

One big flaw I find in the classroom teaching environment is that the pace is different for different people. Some people can manage with a quick explanation, while others want a longer more in depth explanation

Past, Present, and Future

Past (What we've done)

  • Create a database for math problems
  • Create a system where users can ask and answer problems with each other
  • Enable a computer to sort problems into certain skills

Present (What's in progress)

  • Creating a way to add problems to sets and study or share certain problems
  • Open sourcing a hard-coded math problem generator
  • Create a rewards system for those who contribute to the site

Future (What's going to happen)

  • Create a system for open-sourcing lessons
  • Create a math problem generator that generates problems using machine learning
